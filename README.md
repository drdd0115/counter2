# プロジェクト名
Counter2

## 概要
このアプリは、カウント加算・減算するアプリです。
ローカルで作成した個人開発プロジェクトで、GitHubでソースコードを管理しています。

## 作成目的
- JavaScript の学習
- 実践的なアプリ開発経験を積むため

## 主な機能
- カウントアップ
- カウントダウン
- 増減値の変更
- 上限・下限の設定
- 長押し連打

## 使用技術
- HTML
- CSS
- JavaScript
- Git / GitHub
- VSCode

## 画面イメージ
必要に応じてスクリーンショットを追加

## 工夫した点
- シンプルで見やすいUI
- 直感的に操作できる構成
- 増減値の変更や上限・下限など、実用性を意識した機能追加
